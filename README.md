React Testing
=====================
Testing various things:
- React Router
- Props, State, all the react fun
- JSON API mocking

To run locally:
- requires npm / webpack
- npm install
- npm start

to test locally with heroku:
- requires heroku toolbelt / npm / webpack
- npm install
- heroku local web

To deploy, eg to heroku:
- npm run build
- commit
- push to heroku
